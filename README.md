# Search Engine

Поисковый движок для корпоративного портала, разработанный в рамках курса "Разработчик на C++ с нуля".

## Описание

Консольное приложение для поиска по текстовым файлам с ранжированием результатов по релевантности. Поддерживает многопоточную индексацию документов и конфигурацию через JSON-файлы.

## Технологии

- C++20
- CMake 3.20+
- Google Test (юнит-тестирование)
- nlohmann/json (работа с JSON)
- Многопоточность (std::thread)

## Структура проекта

- config/ # Конфигурационные файлы
- include/ # Заголовочные файлы
- resources/ # Текстовые файлы для поиска
- src/ # Исходный код
- tests/ # Модульные тесты


## Сборка и запуск

### Требования

- CMake 3.20 или выше
- Компилятор с поддержкой C++20 (GCC, Clang, MSVC)
